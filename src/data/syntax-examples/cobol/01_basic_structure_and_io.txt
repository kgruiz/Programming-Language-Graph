       IDENTIFICATION DIVISION.
       PROGRAM-ID. HELLOCOBOL.
       AUTHOR. COBOL Learner.

       ENVIRONMENT DIVISION.
       INPUT-OUTPUT SECTION.
       FILE-CONTROL.
           SELECT OPTIONAL USER-INPUT ASSIGN TO KEYBOARD
               ORGANIZATION IS LINE SEQUENTIAL.
           SELECT OPTIONAL USER-OUTPUT ASSIGN TO DISPLAY.

       DATA DIVISION.
       FILE SECTION.
       FD  USER-INPUT.
       01  INPUT-RECORD         PIC X(80).
       FD  USER-OUTPUT.
       01  OUTPUT-RECORD        PIC X(80).

       WORKING-STORAGE SECTION.
       01  WS-USER-NAME         PIC X(30) VALUE SPACES.
       01  WS-MESSAGE           PIC X(50).

       PROCEDURE DIVISION.
       MAIN-LOGIC.
           DISPLAY "Enter your name: ".
           ACCEPT WS-USER-NAME FROM USER-INPUT.

           STRING "Hello, " DELIMITED BY SIZE
                  WS-USER-NAME DELIMITED BY SPACES
                  "!" DELIMITED BY SIZE
                  INTO WS-MESSAGE.
           DISPLAY WS-MESSAGE.

           OPEN OUTPUT USER-OUTPUT.
           MOVE WS-MESSAGE TO OUTPUT-RECORD.
           WRITE OUTPUT-RECORD.
           CLOSE USER-OUTPUT.

           STOP RUN.
